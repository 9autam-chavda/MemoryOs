# MemoryOS

## Problem

People save thousands of screenshots,
notes and documents but cannot find
information later.

## Solution

MemoryOS converts screenshots into
searchable knowledge.

Users can upload screenshots,
extract text using OCR,
and search instantly.

## MVP

- Authentication
- Screenshot Upload
- OCR
- Gallery
- Search
- Dashboard
